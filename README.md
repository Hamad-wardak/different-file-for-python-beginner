## API Reference

#### Get all items

```http
  GET /api/items
```

| parametr | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `` | `string` | **Required**. Your API key |

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
